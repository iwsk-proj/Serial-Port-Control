# IWSK Project
# Prosta aplikacja desktopowa w Pythonie z interfejsem graficznym (Tkinter) do konfiguracji, monitorowania i komunikacji przez port szeregowy (RS-232). Wykorzystuje bibliotekÄ™ `pyserial`.


## Setting up and running
Open a terminal, navigate to the main directory and execute the following commands:
>1. python -m venv env
>2. env\Scripts\activate
>3. pip install -r requirements.txt
>4. python main.py
